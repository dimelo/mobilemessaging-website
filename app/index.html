What is Dimelo Mobile Messaging SDK ?

Dimelo Mobile Messaging SDK (MMS) is our mobile customer care solution.

Dimelo MMS is embedded as a library into native mobile application to provide for rich in-app communication between a brand and its customer.

To enable a seamless customer experience and a streamlined customer service workflow, Dimelo MMS is connected to our multichannel contact center solution Dimelo SMCC.

How can I integrate MMS in my mobile application ?

We provide MMS as an external library to embed into your native mobile application. For iOS this is available on github as a CocoaPods or a static library.

We integrate with the native mobile notification channel for asynchronous conversation and let you customize the visuals, how the user is authenticated and how to trigger a conversation.

As a integrator you just have to load the Dimelo MMS library in your app with an API token, plug it to app notification and decide where/how the user will be able to open it and under which identity

Your first MMS-enabled app is just one API key away

As a developer you can start evaluating Dimelo Mobile Messaging SDK immediately, you just need an API Key, all the documentation is available online.

Go check the iOS getting started instructions to discover how to embed MMS in your own application. For advanced usage and customization have a look at the iOS API reference.

We also provide iOS sample app source code for a quick demonstration of an MMS integration.
